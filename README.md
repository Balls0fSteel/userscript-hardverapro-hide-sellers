# ha-hide-sellers
Simple userscript to hide user(s) from [hardverapro.hu](https://hardverapro.hu)

I love using this site but unfortunately, there are many shady sellers on it as well. This script allows you to hide any/all of them by typing up a list of their usernames on the site.

## Usage:
1. Right-click the currency box in the search results (and optional: type in a note).
2. Use the bottom-right UI to edit your list, notes, etc.
