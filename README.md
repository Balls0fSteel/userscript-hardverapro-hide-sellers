# ha-hide-sellers
Simple userscript to hide user(s) from [hardverapro.hu](https://hardverapro.hu)

I love using this site but unfortunately, there are many shady sellers on it as well. This script allows you to hide any/all of them by typing up a list of their usernames on the site.

## Usage:
1. Check the profile URL of the user you want to hide.  
   Example: `https://hardverapro.hu/tag/TARGET_NAME.html`  
   Use the `TARGET_NAME` to block someone.
   
2. Alternatively, you can right-click a user from the list to add them to the block list.

3. To bypass the block prompt and access the regular menu, hold **Shift** while right-clicking on a username.
