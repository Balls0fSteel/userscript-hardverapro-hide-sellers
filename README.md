# ha-hide-sellers
Simple userscript to hide user(s) from hardverapro.hu

I love using this site but unfortunately there are many shady sellers on it as well. The script allows you to hide any/all of them by typing up a list of their usernames on the site.

## Usage:
Check the profile URL of the user you want to hide. They look like this: https://hardverapro.hu/tag/TARGET_NAME.html
You need to use the TARGET_NAME to block someone. Or you can also right-click the user from the list to add to list.
To bypass the block prompt and access the regular menu, hold shift when you press right-click on a username.
